# Jack HTML Demo

This document outlines how to implement continuous integration using docker, GitHub, Jenkins, Grunt, and Artifactory. The following demo goes through the process of setting up continuous integration for HTML, CSS and JavaScript.

### [Documentation](https://jill.hc-sc.gc.ca/confluence/display/JO/Jack+demo%3A+HTML)

  3) [OptimizeCSSAssetsPlugin](https://github.com/NMFR/optimize-css-assets-webpack-plugin)
